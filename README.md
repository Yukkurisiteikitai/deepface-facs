# Facial Action Coding System(FACS)
顔に現れる特徴(アクションユニット->Action Unit)を使ってどんな表jほうになるのかを決定してその状態を扱うようにする
とりあえず、デモとして扱えるよううにするためには
Facil Action Coding SystemのAction Unitを定義して、その定義に入った形のやつをどうにかしてけんちできるようにしないといけん。
となると必要なのは、Action UnitとそのUnitが実際の顔においてっていうか比率としてどれくらい変わったのかをちゃんとパーソナライズして調節できるようにするっていうことをしないといけないと思うけど、そのUnitってどうやったらできるんかな。
